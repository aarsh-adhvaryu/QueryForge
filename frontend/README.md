# frontend/

React demo UI (stage A7).

Planned features (from the project proposal):
- **Image upload** (from device or URL) to search by image.
- **Catalog browse** — click any product to find visually similar ones.
- **Results grid** — top-10 similar products with image, name, category, similarity score, price.
- **Performance metrics** — show real query latency (ms) to showcase the engine's speed.

Scaffolded against mock data first; wired to the FastAPI backend once it exists.
